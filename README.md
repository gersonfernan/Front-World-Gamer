# Front-World-Gamer
## Installation

> We recommend you use yarn

1. Install all packages

   ```bash
   yarn

   # npm install [for npm]
   ```

2. Run development server

   ```bash
   yarn start

   # npm run start [for npm]
   ```

3. Generate build files for deployment

   ```bash
   yarn  android
   yarn  ios

   # npm run android [for npm]
   # npm run ios [for npm]
   ```
